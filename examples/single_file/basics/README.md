# Basics Examples

Start here for first contact with Arden syntax, variables, and control flow.

Recommended order:

1. `01_hello/01_hello.arden`
2. `02_variables/02_variables.arden`
3. `03_math/03_math.arden`
4. `04_control_flow/04_control_flow.arden`

Run all basics quickly:

```bash
for f in examples/single_file/basics/*/*.arden; do arden run "$f"; done
```
