# Stdlib and System Examples

Focused runtime examples for common standard modules.

Coverage:

- files (`File.*`)
- time (`Time.*`)
- system (`System.*`)
- conversions (`to_*`)
- args (`Args.*`)
- strings (`Str.*`)
- ranges (`Range<T>`)

Run all stdlib/system examples:

```bash
for f in examples/single_file/stdlib_and_system/*/*.arden; do arden run "$f"; done
```
